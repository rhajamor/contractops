# ContractOps Brand Kit

This folder contains the downloadable branding package for the ContractOps website.

## Included Assets

- `contractops-logo-mark.svg` - primary square logo mark
- `contractops-wordmark-dark.svg` - dark theme wordmark lockup
- `contractops-wordmark-light.svg` - light theme wordmark lockup
- `contractops-brand-tokens.json` - color and typography tokens

## Quick Usage

- Product UI icon: use `contractops-logo-mark.svg`
- Website/nav branding: use dark wordmark on dark backgrounds
- Presentation decks on light backgrounds: use light wordmark

## Color Core

- Primary Blue: `#5F86FF`
- Primary Strong: `#3F6DFF`
- Accent Teal: `#1FE8D6`
- Deep Background: `#060916`
- Main Text: `#E9EFFF`
